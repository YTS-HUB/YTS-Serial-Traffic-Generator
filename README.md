# YTS-Serial-Traffic-Generator
A powerful Windows desktop tool for generating custom serial traffic, measuring throughput, injecting errors and verifying COM port hardware — built for engineers working with RS-232 and RS-485 devices

# Description
The YTS Serial Traffic Generator is an easy-to-use desktop tool designed for Windows PCs, allowing users to generate and monitor traffic over serial ports for testing and diagnostics purposes.

## Features

### Serial Communication
- COM Port Communication
- RS232 / RS485 Testing
- Configurable Baud Rate
- Configurable Parity / Stop Bits / Flow Control

### Traffic Generation
- Custom Packet Builder
- ASCII / HEX Transmission
- Binary Data Transmission
- Automatic Packet Repetition
- Timed Traffic Generation
- Randomized Traffic Patterns

### Monitoring & Diagnostics
- Real-time Traffic Monitoring
- RX/TX Statistics
- Timestamped Logging
- HEX + ASCII Display
- Error Detection
- Communication Debugging

### Productivity Features
- Session Logging
- Saved Profiles
- High-performance Traffic Renderer

### Engineering Use Cases
- Embedded Systems Development
- PLC Diagnostics
- Industrial Equipment Testing
- UART Debugging
- Protocol Reverse Engineering
